# Spying user behavior

Debugging Spy provides a tool which spies the Pharo IDE and allows to get data about user's interaction with the Pharo IDE.

If you want to use the tool, please check the [user guide section](./user_guide.md).

In case you want to modify the tool, for example by adding new features, you can read the [documentation](./dev_documentation.md).

To add your contributions in the project, feel free to make a pull request to the [github repository](https://github.com/Pharo-XP-Tools/DebuggingSpy).
